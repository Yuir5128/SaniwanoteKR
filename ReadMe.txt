【설치 방법】

1. 크롬 주소 표시줄에 [chrome ://extensions/]을 입력
또는 주소 표시줄 옆의 … 클릭→[도구 더 보기]→[확장 프로그램] 클릭

2. [압축해제된 확장 프로그램을 로드합니다.]를 클릭
또는 압축 해제한 saniwanoteKR폴더를 드래그 앤 드롭

【제거 방법】

1. 크롬 주소 표시줄에 [chrome ://extensions/]을 입력
또는 주소 표시줄 옆의 … 클릭→[도구 더 보기]→[확장 프로그램] 클릭

2. [사니와수첩] 확장 프로그램의 [세부정보] 항목을 클릭

3. 맨 아래의 [확장 프로그램 삭제]를 클릭

4. 크롬에서 삭제한 후 saniwanoteKR폴더를 삭제
